# GamerZone -  Onde o Jogo Nunca Para!

> "Gamerzone é um site desenvolvido em Next.js, projetado como o seu portal definitivo para notícias, análises e destaques do mundo dos e-sports! Com foco em jogos como CS:GO, Valorant e outros títulos de FPS, o site traz as últimas novidades, resultados de campeonatos, movimentações de times e conteúdos exclusivos para manter você no topo das tendências do cenário competitivo. Aqui, jogadores e fãs encontram tudo o que precisam para acompanhar seus jogos favoritos, descobrir novos talentos e se atualizar sobre as estratégias que dominam o universo dos e-sports."

## Funcionalidades

- Página de Skins: Exibe uma galeria de skins de CS2 e Valorant. Usuários podem visualizar detalhes de cada skin, filtrá-las por tipo e pesquisar por nomes específicos.

- Rankings: Mostra a posição dos times em diferentes Rankings (Ranking da ESL, HLTV, Riot, entre outros).

- Configurações dos Principais PRO-PLAYERS da atualiadade: Na aba "CFG DOS PRO", é dispoinibilizado as principais configurações que os profissionais utilizam no dia-a-dia. Detalhes como mira, configurações de video entre outras, são disponibilizadas

- SEO Otimizado: Páginas otimizadas para SEO utilizando o Next.js, com renderização do lado do servidor para garantir melhor performance e visibilidade nas buscas.

- Responsividade Completa: O site é totalmente responsivo, garantindo uma experiência de navegação fluida em dispositivos móveis e desktop.


## Tecnologias

- **Next.js** - Framework de React que facilita a renderização SSR e SSG
- **React** - Biblioteca de JavaScript para construção de interfaces dinâmicas
- **TypeScript** - Superset de JavaScript que adiciona tipagem estática ao código, melhorando a confiabilidade e manutenção
- **Tailwind CSS** - Biblioteca de classes utilitárias para estilização rápida e responsiva
CSS Modules - Usado para estilização modular e escopo isolado dos estilos por componente
- **PrimeReact** - Biblioteca de componentes UI para React, utilizada para criar interfaces interativas e ricas
- **SwiperJS** - Biblioteca de carrosséis e sliders para exibição de conteúdos de forma deslizante e interativa
- **React Icons** - Biblioteca para fácil inclusão de ícones personalizados e temáticos em componentes

## Pré-requisitos

- **Node.js** versão [versão mínima] ou superior
- **npm** ou **yarn**

## Instalação

1. Clone este repositório:

   ```bash
   git clone https://github.com/dimiendrixmmiranda/gamerzone.git
   cd seu_projeto
   ```

2. Instale as dependências:

   ```bash
   npm install
   ```

   ou

   ```bash
   yarn install
   ```

## Uso

1. Execute o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```

   ou

   ```bash
   yarn dev
   ```

2. Acesse no navegador em [http://localhost:3000](http://localhost:3000).

## Scripts

- `dev`: Inicia o servidor de desenvolvimento.
- `build`: Compila o projeto para produção.
- `start`: Inicia o servidor Next.js em produção.
- `lint`: Verifica e corrige problemas de estilo no código.
